# 需求
设计一个agent

# 输入
1、用户的语音输入（我已经具备的声纹识别模块，你只需要接受识别的结果：用户ID）
2、摄像头图像（可以先用一段视频验证效果）

# 功能清单和工作流程
1、使用多模态模型大模型识别人类（比如男女），主动向用户打招呼（可以夸用户的打扮或者询问有什么问题）
2、无论是否看到人，接收到用户语音后，都要调用声纹识别模块返回的用户ID，查询usr—identity-info文件夹下是否存在文档：用户ID.MD，如果不存在，则新建该文档，并询问用户叫什么,且记录整个任务中用户的行为习惯和偏好，写入该文档中（注意内容一定要简短清晰！）；如果存在，则将用户名字和习惯偏好加入提示词中，并在第一次返回信息里加上对用户的问候，同时记录整个任务中用户的行为习惯和偏好，与原有的信息合并写入该文档中。
3、判别用户意图：如果是闲聊，则陪用户闲聊；如果需要执行任务（具体到执行哪种任务，可以用Fuction calling的方式，我已经写好了很多的mcp工具，但是因为上下文长度，计划先提取用户大致意图，用rag的方式检索前k个送给大模型选择工具）

# 输出
1、如果用户在闲聊，则返回闲聊的文字；如果用户需要你执行任务，则返回：好的，下面我将XXXXX（此处为任务名称）

# 要求
项目里的功能一定要模块化，耦合度尽可能低，方便后面逐模块调试
多模态大模型我已经有了调用python文件，import使用即可
大模型一定尽可能短，将其变短的方法可以作为一个插件或者代码文件实现
大模型的记忆系统也要长远

# 注意
声纹识别和多模态大模型推理模块我已经有了，在两个py文件的函数中，声纹识别返回用户ID ，多模态大模型返回模型输出
